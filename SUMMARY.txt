🆘 TROUBLESHOOTING:

Problem              | Solution
─────────────────────┼─────────────────────────────────────────────────
Docker not found     | Install from docker.com
Port 3001 in use     | Edit docker-compose.yml
MongoDB error        | Check .env file
App won't start      | Check logs: docker-compose logs backend
kubectl not found    | Install Kubernetes cluster
Cluster not found    | kubectl cluster-info

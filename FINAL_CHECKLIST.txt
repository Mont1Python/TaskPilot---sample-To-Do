COMMON ISSUES & FIXES:

Issue: Docker not installed
Fix: Download from https://www.docker.com/products/docker-desktop

Issue: Port 3001 already in use
Fix: Edit docker-compose.yml, change 3001:3001 to 3002:3001
     Access at http://localhost:3002

Issue: Cannot connect to MongoDB
Fix: Check .env file has MONGODB_URI
     Verify internet connection
     Check credentials

Issue: App is blank
Fix: Wait 5-10 seconds for server to start
     Refresh browser
     Check logs: docker-compose logs backend

Issue: Kubernetes pods not starting
Fix: Check logs: kubectl logs pod-name -n todo-app
     Verify image: docker images | grep todo-app
     Check resources: kubectl describe nodes
